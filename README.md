#  Carnet de Voyages Météo

##  Description de l’application

Cette application mobile, développée avec **Ionic + Angular** et utilisant l’API **OpenWeather**, permet de **préparer et organiser des voyages en fonction de la météo**.

L’application combine la **recherche de destinations**, la **consultation météo** et la **création de plannings de voyage** dans une interface simple et intuitive.

---

##  Fonctionnement de l’application

L’application propose **3 actions principales** accessibles via des boutons de navigation.

---

###  1. Rechercher une destination

- L’utilisateur peut rechercher une ville.
- L’application récupère les données météo via **OpenWeather**.
- La ville est ajoutée à la section **Mes destinations**.
- Pour chaque ville, l’utilisateur peut consulter :
  -  Température actuelle
  -  Description météo
  -  Min /  Max
  -  Humidité
  -  Vent
  -  Prévisions sur 5 jours

En cliquant sur une ville, l’utilisateur accède aux **détails météo complets**, incluant les prévisions jour par jour.

---

###  2. Mes destinations (Home)

- Affiche la liste de toutes les villes sélectionnées après recherche.
- Chaque ville montre la météo actuelle et les prévisions sur 5 jours.
- Exemple d’informations affichées :
  - Température actuelle
  - Ciel dégagé / nuageux
  - Min / Max
  - Humidité
  - Vent
  - Prévisions détaillées sur plusieurs jours

---

###  3. Créer un planning de voyage

- L’utilisateur peut créer un **nouveau planning de voyage**.
- Étapes :
  - Donner un nom au planning.
  - Définir une période (date de début et date de fin).
  - Sélectionner une ou plusieurs villes parmi les destinations déjà choisies.
- Un planning peut contenir **une, deux ou plusieurs villes**.

---

###  4. Voir les plannings créés

- L’utilisateur peut consulter la liste de **tous les plannings de voyage**.
- En cliquant sur un planning :
  - Affichage de la description du voyage.
  - Affichage des étapes du voyage selon les villes sélectionnées.
  - Possibilité d’ajouter des **notes pour chaque jour** du voyage.

---

###  Notes journalières

- Pour chaque jour du voyage, l’utilisateur peut :
  - Écrire des notes (activités, idées, impressions, etc.).
  - Sauvegarder les notes.
- Les notes sont conservées et accessibles à chaque ouverture du planning.

---

##  Objectif de l’application

L’objectif de cette application est d’aider l’utilisateur à :
- Choisir ses destinations en fonction de la météo
- Organiser ses voyages facilement
- Suivre les étapes du voyage
- Ajouter des notes jour par jour pour mieux planifier ses activités
